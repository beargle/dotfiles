# SQL Server Management Studio

- Install [SQL Server Management Studio] (SSMS)
- SSMS customizations
    - Reload modified files unless there are unsaved changes
    - Use map mode for vertical scroll bar
    - Show line numbers
    - Show pinned tabs on a separate row
    - Keyboard shortcuts
        - Query
            - Connect `Alt + c`
            - Disconnect `Alt + d`
            - Change connection `Alt + g`
        - SQL Prompt show open tabs `Ctrl + q`
- [SQL Prompt] customizations
    - Format
        - Styles
            - Expand wildcards
            - Qualify object names
            - Remove unnecessary square brackets
    - Tabs
        - Color
            - Production `Red`
            - Staging `Orange`
            - Development `Green`

> **Note**: SSMS and SQL Prompt settings are not automatically configured, and should be manually imported after install.

[SQL Server Management Studio]: https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms
[SQL Prompt]: https://www.red-gate.com/products/sql-development/sql-prompt/
